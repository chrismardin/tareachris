1. Etiquetas de Documento

Las etiquetas de documento definen la estructura básica sobre la cual se construye cualquier página web. Estas etiquetas permiten que el navegador interprete correctamente el contenido y lo presente al usuario.

1.1 <html>

La etiqueta <html> es la raíz del documento y encierra todo el contenido HTML. Incluye el atributo lang, que define el idioma principal de la página.




1.2 <head>

El <head> almacena información esencial que no es visible al usuario, como configuraciones, enlaces externos, metadatos y scripts.



1.3 <body>


Dentro del <body> se encuentra todo el contenido visible por el usuario.


2. Etiquetas de Meta Datos

Estas etiquetas contienen información que ayuda a los navegadores y motores de búsqueda.



2.1 <meta>

Define metadatos como codificación, descripción, autor o palabras clave.



2.2 <style>

Permite insertar reglas CSS directamente dentro del documento.



2.3 <title>

Define el título que aparece en la pestaña del navegador.



2.4 <base>

Establece una URL base para los enlaces relativos.



2.5 <link>

Conecta el documento con recursos externos como hojas de estilo.



3. Etiquetas de Agrupación

Sirven para organizar el contenido dentro de la página.



3.1 <p>

Define un párrafo.



3.2 <hr>

Crea una línea horizontal.



3.3 <ul>

Lista no ordenada.



3.4 <div>

Contenedor genérico.



3.5 <figure>

Encapsula contenido visual.



3.6 <figcaption>



Pie de figura.

4. Etiquetas Textuales

Estas etiquetas modifican el contenido escrito.



4.1 <a>

Crea enlaces internos o externos.



4.2 <span>

Resalta o modifica partes del texto.



4.3 <sub>

Texto en subíndice.



4.4 <sup>

Texto en superíndice.



4.5 <cite>

Identifica el título de una obra.



5. Etiquetas Semánticas

Agregan significado al contenido.



5.1 <nav>

Define una sección de navegación.



5.2 <section>

Representa una sección temática.



5.3 <article>

Contenido independiente.



5.4 <header>

Define encabezados.



5.5 <main>

Contenido principal de la página.



5.6 <footer>

La etiqueta <footer> en HTML es un elemento semántico que define la sección del pie de página de un documento o de una sección específica. Se utiliza para colocar información complementaria como derechos de autor, enlaces a términos legales, datos de contacto, enlaces a redes sociales y otros detalles que no son esenciales para el contenido principal. Es importante recordar que un footer puede aparecer en el pie de página general del sitio o dentro de una sección más específica, como un artículo individual.
