Atributos globales

1.1

id (Identificador Único):
Asigna un identificador único al elemento. Regla: Solo debe haber un elemento con un id específico por página. Uso: Se utiliza para apuntar a elementos desde CSS (#) o JavaScript.

class (Clases de Estilo):
Asigna uno o más nombres de clase al elemento. Regla: Puede haber múltiples elementos con la misma clase. Uso: Selecciona grupos desde CSS (.) o JavaScript.

1.2

style (Estilos en línea):
Permite aplicar estilos directamente al elemento usando propiedades CSS. Regla: Úsalo solo para pruebas rápidas. Uso: style="color: red;".

1.3

title (Información adicional):
Agrega un texto que aparece al pasar el cursor sobre el elemento. Regla: Solo muestra texto simple. Uso: title="Información".

Atributos del formulario (form)

2.1

action (Destino del formulario):
Indica la URL del servidor donde se enviarán los datos.
Regla: Debe ser una ruta válida.
Uso: <form action="procesar.php">.

2.2

method (Método de envío):
Define cómo se enviarán los datos.

Métodos:
• GET → datos visibles en la URL.
• POST → datos ocultos.

Regla: Para datos sensibles usar POST.

Atributos de input

1. type Define el tipo de campo. Ejemplos:

<input type="text"> <input type="number"> <input type="email">


2. name Nombre que recibe el servidor al enviar el formulario. Ejemplo:

<input type="text" name="usuario">


3. placeholder Texto gris que aparece como guía. Ejemplo:

<input type="text" placeholder="Escribe tu nombre">


4. value Valor por defecto. Ejemplo:

<input type="text" value="Hola">
